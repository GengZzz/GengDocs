# 链接
链接文本放在中括号内，链接地址放在后面的括号中，链接title可选。

超链接Markdown语法代码：`[超链接显示名](超链接地址 "超链接title")`

## 基础使用

```sh
这是一个链接 [setvoid.com](https:setvoid.com)
```
:::tip 示例
这是一个链接 [setvoid.com](https:setvoid.com)
:::

## 给链接添加标识

```sh
这是一个链接 [setvoid.com](https:setvoid.com "这是一个链接")
```
:::tip 示例
这是一个链接 [setvoid.com](https:setvoid.com "这是一个链接")
:::
