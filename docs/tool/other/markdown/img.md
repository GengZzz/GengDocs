# 图片
插入图片Markdown语法代码：`![图片alt](图片链接 "图片title")`。

对应的HTML代码：`<img src="图片链接" alt="图片alt" title="图片title">`

## 基本使用

``` sh
![tibet](/img.jpg)
```
::: tip 示例
![tibet](/img.jpg)
:::

## 添加标识
``` sh
![tibet](/img.jpg "拉萨")
```
::: tip 示例
![tibet](/img.jpg "拉萨")
:::

## 链接图片
给图片增加链接，请将图像的Markdown 括在方括号中，然后将链接添加在圆括号中。


``` sh
[![tibet](/img.jpg)](https:setvoid.com)
```
::: tip 示例
[![tibet](/img.jpg)](https:setvoid.com)
:::
