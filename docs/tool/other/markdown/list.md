# 列表
分为有序列表和无序列表

## 有序列表

每个列表项前添加数字并紧跟一个英文句点。数字不必按数学顺序排列，但是列表应当以数字 1 起始。

```sh 
1. 恐龙抗狼
2. 恐龙抗狼
    1. 恐龙抗狼
    2. 恐龙抗狼
3. 恐龙抗狼

```
::: tip 示例
1. 恐龙抗狼
2. 恐龙抗狼
    1. 恐龙抗狼
    2. 恐龙抗狼
3. 恐龙抗狼
:::


## 无序列表

列表项前面添加破折号 (-)、星号 (*) 或加号 (+) 。缩进一个或多个列表项可创建嵌套列表。

```sh 
* 恐龙抗狼
* 恐龙抗狼
    * 恐龙抗狼
    * 恐龙抗狼
* 恐龙抗狼

- 恐龙抗狼
- 恐龙抗狼
    - 恐龙抗狼
    - 恐龙抗狼
- 恐龙抗狼

+ 恐龙抗狼
+ 恐龙抗狼
    + 恐龙抗狼
    + 恐龙抗狼
+ 恐龙抗狼
```

::: tip 示例
* 恐龙抗狼
* 恐龙抗狼
    * 恐龙抗狼
    * 恐龙抗狼
* 恐龙抗狼
:::

## 有序无序嵌套
```sh 
* 恐龙抗狼
* 恐龙抗狼
    1. 恐龙抗狼
    2. 恐龙抗狼
* 恐龙抗狼
```

::: tip 示例
* 恐龙抗狼
* 恐龙抗狼
    1. 恐龙抗狼
    2. 恐龙抗狼
* 恐龙抗狼
:::
