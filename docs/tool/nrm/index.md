# nrm

![nrm iamge](/nrm.png)
[nrm](https://github.com/Pana/nrm)可以帮助我们在不同的镜像间进行切换，帮助你加快软件的下载

## 安装

npm 安装

```sh
npm install -g nrm
```

yarn 安装

```sh
yarn add -g nrm
```

## 常用命令

查看镜像列表

```sh
nrm ls
```

查看当前使用镜像源

```sh
nrm current
```

测试速率

```sh
nrm test
```

选择镜像源

```sh
nrm use taobao
```
