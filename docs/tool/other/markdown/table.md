# 表格
要添加表，请使用三个或多个连字符`---`创建每列的标题，并使用管道`|`分隔每列。您可以选择在表的任一端添加管道。

## 基本使用

``` sh
| 语言 | 简介|
|------|-----|
| java| 这是一段描述|
| HTML| 这是一段描述|
| javaScript| 这是一段描述|
| golang| 这是一段描述|
```

::: tip 示例
| 语言 | 简介|
|------|-----|
| java| 这是一段描述|
| HTML| 这是一段描述|
| javaScript| 这是一段描述|
| golang| 这是一段描述|
:::

## 对齐

在标题行中的连字符的左侧，右侧或两侧添加冒号`:`，将列中的文本对齐到左侧，右侧或中心。

``` sh
| 语言 | 简介 | 其他|
|:------|-----:| :-----------:|
| java| 这是一段描述|其他|
| HTML| 这是一段描述|其他|
| javaScript| 这是一段描述|其他|
| golang|这是一段描述|其他|
```
