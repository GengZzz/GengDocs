# Markdown段落

创建段落没有特殊的格式要求，可以使用空白行对一行或者多行隔离

## 基本使用

```sh

我不去想是否能够成功

既然选择了远方

便只顾风雨兼程
                ---汪国真
```
::: tip 示例

我不去想是否能够成功

既然选择了远方

便只顾风雨兼程      

 ---汪国真
:::

## 换行
在一行的末尾添加两个或多个空格，然后按回车键,即可创建一个换行(`<br>`)

::: warning 不推荐
还有两种其他方式我并不推荐使用。CommonMark 和其它几种轻量级标记语言支持在行尾添加反斜杠 (`\`) 的方式实现换行，但是并非所有 Markdown 应用程序都支持此种方式，因此从兼容性的角度来看，不推荐使用。并且至少有两种轻量级标记语言支持无须在行尾添加任何内容，只须键入回车键（enter）即可实现换行。
:::

## 粗体
要加粗文本可以在单词或短语的前后各添加两个星号`*`或下划线`_`。

``` sh

It's **blog** text
It's __blog__ text

```

::: tip 示例
It's **blog** text  

It's __blog__ text
:::

## 斜体
要加斜体文本可以在单词或短语的前后各添加一个个星号`*`或下划线`_`。

``` sh

It's *Italic* text
It's _Italic_ text

```

::: tip 示例
It's *Italic* text  

It's _Italic_ text
:::

## 粗体和斜体
要同时用粗体和斜体突出显示文本，请在单词或短语的前后各添加三个星号或下划线。要加粗并用斜体显示单词或短语的中间部分，请在要突出显示的部分前后各添加三个星号，中间不要带空格。

``` sh

It's ***Italic and bold*** text
It's ___Italic and bold___ text

```

::: tip 示例
It's ***Italic and bold*** text  

It's ___Italic and bold___ text
:::

## 引用

在段落前加一个`>`

```sh 
>我不去想是否能够成功，既然选择了远方，便只顾风雨兼程。
```

::: tip 示例
>我不去想是否能够成功，既然选择了远方，便只顾风雨兼程。
:::

## 多段落嵌套引用
```sh 
>我不去想是否能够成功，既然选择了远方，便只顾风雨兼程。
>>我不去想是否能够成功，既然选择了远方，便只顾风雨兼程。
```
::: tip 示例
>我不去想是否能够成功，既然选择了远方，便只顾风雨兼程。
>>我不去想是否能够成功，既然选择了远方，便只顾风雨兼程。
:::

## 分隔线

``` sh
***
---
```

::: tip 示例
***
---
:::

## 删除线

单词前后使用两个波浪号`~~`

``` sh
~~delete~~ success

```

::: tip 示例
~~delete~~ success
:::
